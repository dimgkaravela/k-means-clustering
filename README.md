# K-means Clustering

Implementation and analysis of the K-means clustering algorithm for unsupervised machine learning.

## Overview

This project implements the K-means clustering algorithm, a popular unsupervised machine learning method used to group data points into clusters based on similarity.

The goal of the algorithm is to partition a dataset into `K` clusters by iteratively assigning each data point to the nearest centroid and then updating the centroids based on the assigned points.
